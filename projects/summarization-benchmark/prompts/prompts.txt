## [P_EVAL] llm_judge
strategy: llm_as_judge
usage: evaluation_only — do NOT use as generation prompt

You are an expert at evaluating text summaries. Score the following summary
of a given article on these dimensions. Respond ONLY with a JSON object.

ORIGINAL TEXT:
{{ORIGINAL}}

SUMMARY TO EVALUATE:
{{SUMMARY}}

Score each dimension from 1-5:

- faithfulness: Does the summary accurately reflect the original? (1=many errors, 5=perfectly accurate)
- coverage: Does it capture the key information? (1=misses most, 5=captures all key points)
- conciseness: Is it appropriately brief? (1=too long/short, 5=perfect length)
- fluency: Is it well-written and natural? (1=awkward, 5=excellent prose)
- coherence: Is it logically organized? (1=confusing, 5=very clear)

Respond with ONLY this JSON, no other text:
{
“faithfulness”: <score>,
“coverage”: <score>,
“conciseness”: <score>,
“fluency”: <score>,
“coherence”: <score>,
“overall”: <average>,
“brief_rationale”: “<one sentence explaining the overall score>”
}
